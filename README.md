# Real Cloud Threat Visualizer

A **real-time cybersecurity monitoring dashboard** that visualizes simulated cloud attack events (brute force, phishing, port scans) across fake endpoints built using **Node.js, Socket.IO, React, Tailwind CSS, and Recharts**.

---

##  Features
- Real-time simulated attack data stream
- WebSocket communication (Socket.IO)
- Interactive live charts and attack logs
- Animated topology map of fake cloud endpoints
- 100% local & cost-free (no AWS/Google credentials)

---

## Skills Demonstrated
- Cloud infrastructure simulation  
- Event-driven architecture  
- Real-time visualization (WebSockets + React)  
- Frontend design with Tailwind CSS & Recharts  
- Secure data handling principles  

---

| Area | Tools |
